Notes

  • Hover.css is pretty easy to plug in, but some theme styles are built-in - e.g. Bounce To Right always uses its own blue colour highlight by default. I have to see where this is specified and how to customise it.
  • This library is great for study and/or reverse-engineering the raw CSS mechanisms.
  • We can combine Hover.css classes for fun and profit, but not all combinations work out of the box. (c.f. Grow+Bounce vs Grow+Float)
